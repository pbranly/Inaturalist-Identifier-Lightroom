🐾 Présentation

Le plugin iNaturalist Identifier s’intègre à Adobe Lightroom Classic pour aider les photographes à identifier automatiquement les espèces visibles sur leurs clichés. Il utilise l’API iNaturalist pour analyser les images sélectionnées et ajoute des balises d’identification selon les résultats obtenus.

✨ Fonctionnalités

  • 📤 Envoi des photos vers l’API de reconnaissance d’image d’iNaturalist
  • 🏷️ Balises automatiques avec les espèces identifiées
  • 📚 Traitement en lot des photos (fonctionnalité à venir)
  • 🔍 Affichage des réponses détaillées de l’API pour chaque image
  • 🧩 Plugin léger et simple, développé en Lua

⚙️ Installation

  1. 📥 Télécharger la dernière version
  2. Ouvrir Lightroom Classic
  3. Aller dans Fichier > Gestionnaire de modules externes
  4. Cliquer sur Ajouter et sélectionner le dossier du plugin
  5. Confirmer l’installation

🔐 Configuration

Le plugin nécessite un jeton d’accès API iNaturalist :

  • Créer un compte sur iNaturalist
  • Générer un jeton personnel dans les paramètres du profil
  • Entrer le jeton dans le panneau du plugin dans Lightroom

⚠️ Remarque : Les jetons sont valides pendant 24 heures seulement.

🖼️ Utilisation

  • Sélectionner une ou plusieurs photos dans la bibliothèque Lightroom (le mode lot n’est pas encore entièrement testé)
  • Utiliser le menu du plugin pour les envoyer à l’API iNaturalist
  • Examiner et appliquer les balises d’espèces selon les résultats
